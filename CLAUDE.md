# Orc

macOS native agent multiplexer. Rust core + SwiftUI.

## Stack

- Swift 6 + SwiftUI (macOS 14+)
- Rust (cargo workspace, edition 2024)
- UniFFI (Rust ↔ Swift FFI)
- Xcode + xcodegen

## Structure

```
crates/        rust workspace members (core, agent, ffi)
apple/         swift app (App, Models, Store, Views, Bridge, Resources)
bindings/      uniffi generated (gitignored)
```

## Commands

```
cargo check                    # check rust
xcodegen generate              # regenerate xcodeproj from project.yml
xcodebuild -scheme Orc build   # build app
```

## UI Hierarchy

Workspace → Tab → Panel (recursive tree) → Surface (terminal/agent/browser)

- Panel is a tree: leaf (has Surface) or split (has children)
- Surface is independent from Panel — swappable between slots

## Gotchas

- `Orc.xcodeproj` is generated by xcodegen — edit `project.yml` instead
- `bindings/` is gitignored — generated by uniffi at build time
- `target/` is gitignored — cargo build output
